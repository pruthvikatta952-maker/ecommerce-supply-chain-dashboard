# E-commerce Supply Chain Performance Dashboard

## Project Overview

This project is an end-to-end Power BI dashboard designed to analyze and monitor e-commerce supply chain operations. The dashboard focuses on shipment performance, order fulfillment, carrier efficiency, freight cost trends, inventory health, and return analysis.

The objective of this project was to simulate real-world logistics and supply chain reporting scenarios commonly used in retail, distribution, and e-commerce operations.

---

# Tools & Technologies

- Power BI
- DAX
- Power Query
- Microsoft Excel

---

# Business Objectives

The dashboard was created to answer key supply chain and logistics questions such as:

- Are customer orders being delivered on time?
- Which carriers have the highest delays?
- Which delivery modes are driving freight costs?
- Which products are below reorder point?
- Which categories have the highest return rates?
- How is inventory distributed across warehouses?

---

# Dashboard Pages

## 1. Executive Overview
Provides a high-level summary of operational KPIs including:
- Total Orders
- Total Shipments
- Total Freight Cost
- Cost Per Shipment
- On-Time Delivery %
- OTIF %
- Fill Rate %
- Return Rate %

### Key Visuals
- Orders vs Shipments Trend
- Shipment Status Breakdown
- Freight Cost by Carrier
- Freight Cost by Province
- Top Delay Reasons

---

## 2. Carrier Performance
Analyzes logistics carrier efficiency and delivery performance.

### Key Visuals
- On-Time Delivery % by Carrier
- Average Delivery Days by Carrier
- Delayed Shipments by Carrier & Delay Reason

---

## 3. Fulfillment & Returns
Tracks customer fulfillment quality and return trends.

### Key Visuals
- Fill Rate % by Category
- Return Rate % by Category
- Total Orders by Sales Channel

---

## 4. Inventory Health
Monitors inventory levels and stock risks across warehouses.

### Key Visuals
- Current Stock by Product
- Inventory Value by Category
- SKUs Below Reorder Point

---

## 5. Cost Analysis
Analyzes freight spending and transportation costs.

### Key Visuals
- Cost Per Shipment by Carrier
- Freight Cost by Delivery Mode
- Freight Cost Trend by Month

---

# Key KPIs

| KPI | Description |
|---|---|
| OTIF % | Orders delivered On-Time and In-Full |
| Fill Rate % | Percentage of ordered quantity fulfilled |
| On-Time Delivery % | Percentage of shipments delivered on time |
| Return Rate % | Percentage of returned customer orders |
| Cost Per Shipment | Average freight cost per shipment |
| Inventory Below Reorder Point | Products requiring replenishment attention |
